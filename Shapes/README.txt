A Menu Driven program to calculate the Area and Volume of the selected Shapes.
It has different classes as Circle, Rectangle, Square, Sphere, Cylinder, and Pyramid.
An abstract class called Shape with showShape(String shape) as non-abstract method, with calculateShape() and calculatePerimeter() as abstract method.
An interface called Volume with calculateVolume() as an abstract method.
It takes input from users for measurements of shapes and prints the result.
