An application to show the behavior of a Duck.

1) Fly Behaviour :
   a) <<interface>> FlyBehavior
   b) Two different classes  FlyWithWings and FlyNoWay which implement FlyBehavior containing fly() method.
2) Quack Behaviour :
   a) <<interface>> QuackBehavior
   b) Two different classes  Quack and Squeak which implement QuackBehavior containing quack() method.
3) Swim Behaviour :
   a) <<interface>> SwimBehavior
   b) Two different classes - Swim and NoSwim which implement SwimBehavior containing swim() method.

Printing which duck will fly, quack or swim.

Client : RedHeadDuck, DecoyDuck
